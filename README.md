## Développement local

Le site est entièrement statique : il suffit d'ouvrir `index.html` dans un navigateur, ou de servir le dossier avec un serveur local.

```bash
# exemple avec Python
python3 -m http.server 8000
# puis ouvrir http://localhost:8000
```
